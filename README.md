# 无宁县志 (Wuning County)

一个基于文本的古风模拟经营与角色扮演游戏。在这座名为“无宁”的县城中，你可以扮演不同的角色，体验不同的人生轨迹，影响县城的命运。

## 🎮 游戏简介

《无宁县志》是一款纯文本交互的网页游戏。游戏以天为单位进行，玩家需要管理自己的体力、金钱、声望等资源，通过日常工作、社交互动、设施游玩等方式推动游戏进程。

## ✨ 核心特色

### 🎭 多重身份体验
游戏目前提供三种截然不同的开局身份，每个身份都有独特的属性、技能和玩法重心：

*   **🏛️ 县丞 (Magistrate)**
    *   **定位**：辅助县令，治理一方。
    *   **核心玩法**：颁布政令（如休养生息、严整治安）、巡视乡里。
    *   **目标**：平衡县城的各项指标（民生、治安、经济、文化），积累声望。
*   **💰 商人 (Merchant)**
    *   **定位**：长袖善舞，经营致富。
    *   **核心玩法**：风险投资、商业经营。
    *   **优势**：初始资金雄厚，日常工作收益高。
*   **⚔️ 少侠 (Hero)**
    *   **定位**：行走江湖，快意恩仇。
    *   **核心玩法**：闭关修炼、行侠仗义。
    *   **特色**：拥有极高的体力和武学潜力，可参加武林大会。

### ⚙️ 丰富游戏系统

1.  **📅 日常循环**
    *   **工作与休息**：每日行动次数有限，需合理安排劳逸结合。
    *   **天气系统**：每日随机天气（晴、雨、雪），不同天气会触发特定事件（如雨天探索可能获得“被诅咒的剑”）。
    *   **健康机制**：体力过低会影响工作，健康状态决定每日恢复量。

2.  **🏪 深度经济系统**
    *   **市场波动**：粮草、布匹、茶叶等商品价格每日随供需波动，考验玩家“低买高卖”的商业头脑。
    *   **产业置办**：可购买良田、商铺等产业，获得每日被动收入（受县城经济繁荣度加成）。

3.  **🌲 养成与成就**
    *   **天赋树**：消耗“阅历”提升天赋，永久增强角色属性（如“经商之道”、“强身健体”）。
    *   **成就系统**：达成特定条件（如“第一桶金”、“夜雨江湖”）解锁成就并获得大量阅历。

4.  **🗣️ NPC 交互**
    *   结识各色人物，与他们交谈、送礼建立羁绊。
    *   注意：话说多了嗓子会哑（每日聊天有上限），导致第二天无法说话！

5.  **🎪 游乐坊设施**
    *   **🎯 无宁箭馆**：消耗体力与金钱练习射箭，考验武学能力，赢取彩头。
    *   **🎲 小司赌坊**：三颗骰子定输赢，运气与胆量的博弈。
    *   **🔮 天机阁**：每日一签，求问吉凶，指点迷津（每日限一次）。

6.  **📜 政令系统 (县丞专属)**
    *   消耗声望颁布持续性政令，深远地影响县城的发展方向。

## 🛠️ 技术栈

本项目使用现代前端技术构建：

*   **核心框架**: [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
*   **构建工具**: [Vite](https://vitejs.dev/)
*   **样式方案**: [Tailwind CSS](https://tailwindcss.com/) (支持响应式布局)
*   **状态管理**: [Zustand](https://github.com/pmndrs/zustand) (包含中间件实现数据持久化)
*   **路由管理**: [React Router](https://reactrouter.com/)
*   **图标库**: [Lucide React](https://lucide.dev/)

## 🚀 快速开始

### 环境要求
*   Node.js (推荐 v16+)

### 安装与运行

1.  **克隆项目**
    ```bash
    git clone <repository-url>
    cd Wuning-County
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **启动开发服务器**
    ```bash
    npm run dev
    ```
    打开浏览器访问控制台输出的地址（通常是 `http://localhost:5173`）。

4.  **构建生产版本**
    ```bash
    npm run build
    ```
